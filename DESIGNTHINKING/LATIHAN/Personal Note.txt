	| roleid | rolename     | description                               |
	| ------ | ------------ | ----------------------------------------- |
	| 1      | owner        | Hak akses penuh, pengelola utama aplikasi |
	| 2      | admin        | Mengelola user, stok, laporan             |
	| 3      | kasir        | Mengelola transaksi penjualan             |

Additional Description
----------------------
1. Owner (Pemilik)
Deskripsi: Pengguna dengan hak akses penuh, pengelola utama aplikasi dan usaha.
Hak akses:
Kelola user dan role (termasuk owner lain)
Lihat dan kelola semua data (stok, keuangan, laporan)
Setting aplikasi dan konfigurasi penting

2. Admin
Deskripsi: Pengelola operasional harian, membantu owner dalam mengelola data.
Hak akses:
Tambah/edit/hapus user selain owner
Kelola stok dan produk
Lihat dan buat laporan keuangan dan stok
Tidak bisa mengubah role owner

3. Kasir
Deskripsi: Bertugas mengelola transaksi penjualan.
Hak akses:
Proses transaksi penjualan dan pembayaran
Lihat stok produk terbatas
Input data penjualan harian
Tidak bisa mengubah data user dan role

4. Staf Gudang
Deskripsi: Mengelola persediaan barang masuk dan keluar.
Hak akses:
Input barang masuk dan keluar stok
Lihat data stok secara lengkap
Tidak bisa mengakses data keuangan dan user

5. Manager
Deskripsi: Memantau dan memonitor kinerja usaha dan laporan.
Hak akses:
Lihat laporan keuangan dan stok lengkap
Lihat data penjualan
Tidak bisa mengubah data user, stok, atau keuangan
